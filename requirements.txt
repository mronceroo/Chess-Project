# ============================================
# NOTES
# ============================================
# 
# Installation:
# 1. Create virtual environment:
#    python -m venv venv
#    source venv/bin/activate  # Linux/Mac
#    venv\Scripts\activate     # Windows
#
# 2. Install requirements:
#    pip install -r requirements.txt
#
# 3. Install FFmpeg separately (see above)
#
# 4. Download and setup LM Studio:
#    - Download from https://lmstudio.ai/
#    - Download Hermes 3 Llama 3.2 - 3B model
#    - Start local server on port 1234
#
# 5. On first run, models will be downloaded:
#    - YOLOv8 weights (~6MB)
#    - Whisper model (~1.5GB for medium)
#    - SpeechT5 embeddings (~400MB)
#
# GPU Support:
# - For CUDA 11.8: Use torch 2.0.0+cu118
# - For CUDA 12.1: Use torch 2.0.0+cu121
# - Check: https://pytorch.org/get-started/locally/
#
# Minimum System Requirements:
# - RAM: 8GB (16GB recommended with LLM)
# - Storage: 5GB free space
# - Webcam or external camera
# - Microphone (built-in or external)
